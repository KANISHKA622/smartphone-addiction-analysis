# 📱 Smartphone Usage & Addiction Analysis Dashboard
 🚀 Project Overview

The **Smartphone Usage & Addiction Analysis Dashboard** is an **AI-assisted analytics project** created to analyze smartphone usage behavior across **1,200 users**.

This dashboard converts raw behavioral data into actionable visual insights to understand how smartphone habits affect:

✔ Sleep patterns
✔ Stress levels
✔ Productivity
✔ Digital addiction risk

The project demonstrates the ability to transform complex data into a clean and professional analytical dashboard.

🎯 Business Objective

The goal of this project is to:

* Identify excessive smartphone usage patterns
* Detect digital addiction indicators
* Analyze the impact on mental wellness
* Compare behavioral trends across demographics
* Present insights through visual storytelling

 📊 Dashboard Features
 KPI Metrics

* Total Users
* Average Screen Time
* Average Sleep Hours
* Addiction Rate
* Notifications per Day
* App Opens per Day

 Visual Insights

* Gender distribution analysis

* Addiction labels
  
💡 Key Insights

Some important findings from the dashboard:

* Higher screen time is linked to lower sleep duration
* Social media usage strongly influences addiction risk
* Frequent notifications increase app engagement
* Stress levels rise with excessive smartphone usage
* Productivity declines among high-risk users
